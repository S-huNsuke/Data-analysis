# Data Analysis

基于 Python 的数据分析项目集合。

## 项目列表

### 1. 天气数据可视化
- 数据获取与可视化展示
- 24小时/7天温度、湿度、风力趋势分析

### 2. B站UP主数据可视化
- B站榜单数据分析
- 词云生成、树状图可视化

### 3. 游戏销售数据分析
- 全球游戏销售数据统计与可视化

### 4. 电商用户特征数据分析
- 用户个性化特征分析

## 技术栈

- Python
- pandas
- matplotlib
- seaborn

## 使用方法

各子项目目录下均有独立的 README.md 文件，请参阅具体项目的说明。
