# 电商用户特征数据分析

## 项目简介
基于Python的电商用户特征数据分析工具，通过对用户个性化特征数据的深入分析，帮助企业了解用户行为模式、消费习惯和价值分层，为精准营销和用户运营提供数据支持。

## 功能特性
- **数据处理与清洗**：数据格式转换与标准化、中文列名与值映射、缺失值处理
- **用户分层分析**：
  - 年龄群体划分：少年、青年、中年、中老年、老年
  - 收入群体划分：低收入、中低收入、中高收入、高收入
  - RFM模型分析：基于最近登录时间(R)、消费金额(M)、购买频率(F)的用户价值分层
- **多维度用户特征分析**：性别分布、年龄群体与性别交叉分析、位置分布、兴趣分布、订阅活动通知对用户行为的影响
- **消费行为分析**：整体客单价计算、订阅用户与未订阅用户客单价对比、购买频率与消费金额关系分析
- **数据可视化**：饼图（性别分布、兴趣分布）、柱状图（位置分布、年龄群体性别分布）、交叉分析图表

## 技术栈
- pandas, numpy
- matplotlib, seaborn

## 项目结构
```
e-commerce-user-analysis/
├── main.py                          # 主程序入口
└── user_personalized_features.csv   # 用户特征数据文件
```

## 使用方法

```bash
# 安装依赖
uv sync

# 运行分析程序
uv run python e-commerce-user-analysis/main.py
```

输出结果保存在 `output/` 目录下。
